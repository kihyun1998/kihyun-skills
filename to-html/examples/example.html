프로젝트 빠른 시작

저장소 클론부터 개발 서버 실행까지 — to-html 변환 예시

목차

  1. 설치
  2. 실행

설치

저장소를 클론하고 의존성을 설치합니다.

git clone https://example.com/repo.git
cd repo
npm install
권장 환경
Node.js 18 이상에서 동작을 확인했습니다.

실행

개발 서버를 띄우고, 필요에 따라 프로덕션 빌드를 만듭니다.

명령설명
npm run dev개발 서버 실행 (자동 새로고침)
npm run build프로덕션 빌드 생성
포트 충돌
기본 포트 3000이 이미 사용 중이면 PORT 환경 변수로 바꿔 실행하세요.
